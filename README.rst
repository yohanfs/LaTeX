Command Line
---------------------------------------------------------------------------------

- basic document

::

      $ pdflatex filename

- include bibtex

::

      $ pdflatex filename
      $ bibtex filename
      $ pdflatex filename
      $ pdflatex filename

- include glossaries

::

      $ pdflatex filename
      $ makeglossaries filename
      $ pdflatex filename
      $ pdflatex filename
